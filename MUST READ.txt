how to use : 

- A button :

<Button title="the title" id="the id" class="buttonPrincipal or buttonSecondaire"/>
- An input : 
2- import Input from "./Composants/Input.js";
<Input label="the title" id="the id" class="any additionnal class"/>

3- userContext : 
  const {user, login, logout} = useContext(UserContext);
user : autUser : gets the user object , IN CACHE : type: if it's an atc or decideur , auth : logged in or not

login: fct that sets context and cache to logged in state (in context : saves the user object, in cache : saves user id)
	TO CALL IT : () => login (userObject, userType)
logout : sets context to a logged out state - sets cache to a logged out state
	TO CALL IT : logout
